# 🔐 Password Generator

A feature-rich Password Generator built with Python that creates strong, secure, and customizable passwords. Users can choose the password length, select character types, generate multiple passwords at once, check password strength, and automatically save generated passwords to a text file.

---

## ✨ Features

- Generate secure random passwords
- Custom password length
- Include uppercase letters (A-Z)
- Include lowercase letters (a-z)
- Include numbers (0-9)
- Include special characters (!@#$%^&*)
- Password strength indicator
- Generate multiple passwords at once
- Automatically save passwords to `passwords.txt`
- Easy-to-use command-line interface
- Input validation for better user experience

---

## 🛠️ Technologies Used

- Python 3
- `random` module
- `string` module
- File Handling
